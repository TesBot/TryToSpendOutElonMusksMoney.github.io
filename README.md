## 🎮 游戏简介

<img width="2549" height="1403" alt="image" src="https://github.com/user-attachments/assets/af33d651-84d5-4ae3-b831-2c639f1f7f24" />

你将获得 **8390亿美元**（埃隆·马斯克的真实身家），挑战在7天内将其全部花光！

游戏包含多种烧钱方式：
- 💥 **炸裂消费** - 私人飞机、天气改造、全球地标广告
- ☕ **日常消费** - 特斯拉定制、私人飞机、房产维护
- 📦 **囤货交易** - 奢侈品、海岛、公司股权
- 🎰 **豪赌博弈** - 赌场、地下赛事、盲盒
- 📈 **风险投资** - 加密货币、科技风投
- 🔥 **丧心病狂** - 太空项目、艺术品销毁、极端基建

## ✨ 特色功能

### 三种游戏模式
| 模式 | 说明 |
|------|------|
| 📅 经典七日记 | 真实计时7天，支持存档/读档 |
| ⚡ 快节奏烧钱 | 可调倍速，短时间内体验全部玩法 |
| ♾️ 无尽烧钱 | 无时间压力，专注操作决策 |

### 时间倍速
- 1x / 100x / 1000x / 10000x / 20000x

### 真实经济系统
- 所有操作都有冷却时间（CD）
- 库存限制与刷新机制
- 价格随购买次数上涨
- 投资有盈亏风险

## 🚀 如何运行

直接在浏览器中打开 `index.html` 即可开始游戏：

```bash
# 克隆仓库后
cd HowToSpendOutMusksMoney
open index.html
```

或使用本地服务器：

```bash
python -m http.server 8080
# 然后访问 http://localhost:8080
```

## ⚠️ 游戏规则

1. ❌ 禁止直接转账、赠与他人
2. ❌ 所有操作都有冷却时间

5. ⏱️ 剩余时间为0时游戏结束
